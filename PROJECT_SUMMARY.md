# YouTube Toolkit - Project Summary

## Overview
A complete Python-based solution for monitoring YouTube channels and extracting video transcripts automatically.

## Core Functionality
✅ Monitor YouTube channels for new videos (last 7 days by default)
✅ Download video subtitles automatically
✅ Convert subtitles to clean, readable text
✅ Organize output by channel in separate folders
✅ Create annotated transcripts with metadata
✅ Support for multiple channels simultaneously
✅ Configurable via JSON or Python scripts

## Key Features

### 1. Automated Channel Monitoring
- Checks channels for videos published in last N days
- Filters recent content automatically
- Processes videos in batch
- Generates summary reports

### 2. Intelligent Text Processing
- Merges overlapping subtitle segments
- Removes duplicate content
- Creates proper paragraphs based on timing gaps
- Maintains readability for both monologues and interviews

### 3. Rich Metadata Annotation
Each transcript includes:
- Video title, URL, and ID
- Publication date and time
- Channel information
- Generation timestamp

### 4. Flexible Configuration
- Multiple channels support
- Customizable lookback period
- Multi-language subtitle support
- Configurable output directories

1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Configure API key in `.env`
4. Edit `monitor_channels.py` or create `channels_config.json`
5. Run: `python monitor_channels.py`
